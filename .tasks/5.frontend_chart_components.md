## Steps

### 1. Create `src/utils/filterByRange.js` ‚Äî time range filtering utility

Client-side filtering of `{ date, rank }` arrays. Each chart manages its own range state.

```js
export function filterByRange(data, range) {
  const cutoffs = { "1M": 30, "6M": 180, "1Y": 365, "3Y": 365 * 3, "ALL": Infinity };
  const days = cutoffs[range];
  if (days === Infinity) return data;
  const cutoff = new Date(Date.now() - days * 86400000);
  return data.filter(d => new Date(d.date) >= cutoff);
}
```

Wraps each chart section with a fade+slide-up animation on scroll.

```jsx

```
