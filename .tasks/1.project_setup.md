## Goal
Bootstrap the full project from scratch: Vite + React app, all dependencies, Tailwind dark mode, Cloudflare config, folder structure, index.html with OG tags, npm scripts, env template, and gitignore.

## Steps

1. **Initialize Vite + React**
   - `npm create vite@latest . -- --template react`
   - Remove boilerplate from `App.jsx` and `App.css`
   - `npm install`

2. **Install dependencies**
   - Production: `npm install tailwindcss @tailwindcss/vite recharts framer-motion`
   - Dev: `npm install -D postcss autoprefixer wrangler`

3. **Configure Tailwind with dark mode**
   - `npx tailwindcss init -p`
   - Set `darkMode: "class"` and `content: ["./index.html", "./src/**/*.{js,jsx}"]` in `tailwind.config.js`
   - Add `@tailwind base/components/utilities` directives to `src/index.css`

4. **Set up `wrangler.toml`**
   ```toml
   name = "didtwitterdie"
   compatibility_date = "2024-01-01"
   compatibility_flags = ["nodejs_compat"]
   pages_build_output_dir = "dist"

   [[kv_namespaces]]
   binding = "CACHE"
   id = "<your-kv-namespace-id>"
   ```

5. **Create project structure**
   ```
   src/
   ├── hooks/
   ├── components/
   └── utils/
   functions/
   └── api/
   public/
   ```

6. **Set up `index.html`** with OG meta tags, favicon, title "Did Twitter Die?"

7. **Add `package.json` scripts**
   ```json
   "dev": "wrangler pages dev -- vite",
   "build": "vite build",
   "preview": "wrangler pages dev dist",
   "deploy": "wrangler pages deploy dist"
   ```

8. **Create `.dev.vars.example`**
   ```
   CLOUDFLARE_RADAR_TOKEN=your-cloudflare-api-token-here
   ```

9. **Set up `.gitignore`**
   ```
   node_modules/
   dist/
   .dev.vars
   .wrangler/
   *.local
   .DS_Store
   ```

## Acceptance Criteria
- `npm run dev` starts Vite + Wrangler dev server without errors
- Tailwind utility classes render correctly (dark mode toggleable via `class`)
- Project structure matches README spec
- `index.html` has proper OG meta tags and title
- `.gitignore` excludes secrets, build output, and node_modules
- `.dev.vars.example` exists as a template for contributors
