## Problem

The footer attribution for our two data sources (Cloudflare Radar and Tranco List) is incomplete. Research into both sources' licensing terms revealed several gaps that need fixing before launch.

### Research Findings

#### Cloudflare Radar
- **License**: CC BY-NC 4.0 (Creative Commons Attribution-NonCommercial 4.0)
- **Attribution required**: Source name, link to material, link to license, indicate if changes made
- **Trademark rules**: "Cloudflare" and "Cloudflare Radar" are trademarks — use as adjective, capitalize, include trademark attribution statement
- **No logo usage** without written permission (we only use text — compliant)
- **NC restriction**: Cannot use for commercial advantage. If site ever has ads/monetization, need separate permission from radar@cloudflare.com

#### Tranco List
- **No explicit license** on their data or GitHub repo (DistriNet/tranco-list)
- **Academic project** from KU Leuven & Hasselt University
- **Expected attribution**: Citation of their paper — Le Pochat et al., "Tranco: A Research-Oriented Top Sites Ranking Hardened Against Manipulation", NDSS 2019 (DOI: 10.14722/ndss.2019.23386)
- **Upstream licenses carry through**: Tranco aggregates data from:
  - Cloudflare Radar (CC BY-NC 4.0)
  - Chrome UX Report (CC BY-SA 4.0)
  - Majestic (CC BY 3.0)
  - Cisco Umbrella (no formal license)
  - Farsight (no formal license)
- The NC restriction from Cloudflare Radar applies transitively to Tranco's aggregated data
